- **에이전트를 위한 OOP / DDD** — 거대한 프롬프트가 아닌, 작은 스킬·명시적 워크플로 단계·사람의 체크포인트

- **AI 코딩 에이전트 하네스** — 에이전트가 실제로 따를 수 있는 방법론 팩
- **Java & Python 백엔드** — 유스케이스, 애그리거트, 포트, 테스트 가능한 경계
